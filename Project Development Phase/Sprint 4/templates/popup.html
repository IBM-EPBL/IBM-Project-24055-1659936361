About Us

×
The purpose of the web application is to predict the average rainfall in a month. It also provides the suggestions of crops that can be grown. It also provides the information about all the crops like food, feed, fiber, oil, flowers etc. It is a machine learning model with Snap Random Forest algorithm. We have used the best regressor algorithm provide by auto AI feature by the IBM Watson Studio & IBM cloud for storage and deployment. The dataset used is the Indian rainfall data from data.gov.in website. We have used the mot popular Twilio services for the OTP generation and sending of Messages. Python flask Web framework for the Backend purpose and for frontend, HTML, CSS and JavaScript are used. The user data are stored in database created by MySQL. We are open to suggestion for improvement of our project.
